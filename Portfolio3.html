Adobe XD & Processing

Overview

These programs are apps and code created using Adobe XD and processing. The important thing in these works is whether I can acquire knowledge and theory about programs and design that are essential to UX design.

Content

The contents of assignments are as followed:
• Create a prototype of an application that reflects (parts of) your personal identity. (Adobe XD)
• Create a drawing of a thing or a concept which, every time you run the sketch, is different but still the same. (Processing)
• Make free assignment by using elements: Shapes and colours, User input, Conditionals, Functions, Loops, Arrays. (Processing)
